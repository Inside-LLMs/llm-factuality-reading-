# DoLa：Decoding by Contrasting Layers Improves Factuality in LLMs (ICLR 2024)

## 1. 论文要解决什么问题？

### 问题

LLM 经常 hallucination（生成事实错误内容）。

例：
Q: Washington State 的首都？
LLM: Seattle ❌
GT: Olympia ✅

目标：

> 不做微调（fine-tuning）
> 不接外部知识库（RAG）
> 只改 decoding，就提升 factuality

## 2. 作者认为 hallucination 为什么产生？

作者观察：

Transformer 不同层存储的信息不同：

- 早层（early layers）：
  - syntax
  - surface pattern
  - language fluency

- 高层（later layers）：
  - semantics
  - factual knowledge

预测事实时：

错误但“看起来合理”的 token（Seattle）→ 所有层概率都高

真正正确 token（Olympia）→ 后面层概率逐渐升高

因此：

事实知识是在后层逐渐注入（inject）进去。

核心假设：

> 对比高层与低层输出，能放大 factual knowledge。

## 3. 核心方法：DoLa

DoLa = Decoding by Contrasting Layers

不是直接用 final layer 输出：

```text
p(x_t|x_{<t}) = softmax(φ(h_t^{(N)}))
```

其中：

- h_t^{(N)}: final layer hidden state
- φ(·): LM head

DoLa 增加 early exit：

对第 j 层：

```text
q_j(x_t|x_{<t}) = softmax(φ(h_t^{(j)}))
```

得到：

- 成熟层（mature layer）：q_N
- 早期层（premature layer）：q_M

然后做对比：

```text
F(q_N, q_M) = log(q_N(x_t) / q_M(x_t))
```

最终预测：

```text
\hat p(x_t|x_{<t}) = softmax(log(q_N / q_M))
```

## 4. 怎么选 premature layer？

不是固定某层。

作者提出：

### Dynamic Premature Layer Selection

利用 Jensen-Shannon Divergence：

```text
d(q_N, q_j) = JSD(q_N || q_j)
```

选择 divergence 最大层：

```text
M = argmax_{j∈J} JSD(q_N || q_j)
```

即：挑选与最终层差异最大那层。

原因：差异越大 → 后续层注入了更多知识 → 更可能包含 factual update。

## 5. 真正 contrast 怎么做？

类似 Contrastive Decoding：

定义：

```text
F(q_N, q_M) = log( q_N(x_t) / q_M(x_t) )
```

最终：

```text
\hat p = softmax( log(q_N / q_M) )
```

直觉：如果 token 在 final layer 概率高、early layer 概率低，则 q_N/q_M 会变大 → 被强化。

即：后层新增事实知识会被放大。

## 6. Adaptive Plausibility Constraint (APC)

避免 contrast 把垃圾 token 放大：

只保留：

```text
V_head = { x_t : q_N(x_t) ≥ α max_w q_N(w) }
```

否则：

```text
F = -∞
```

论文设定：

```text
α = 0.1
```

作用：避免 false positive / false negative。

## 7. 完整推理流程

输入：Prompt
↓
得到所有 layer logits
↓
计算：q_0, q_2, q_4, …
↓
计算：JSD(q_N, q_j)
↓
选：M = argmax JSD
↓
contrast：log(q_N / q_M)
↓
softmax
↓
生成 token

## 8. 实验结论

### TruthfulQA

LLaMA-7B：

原始：MC2 = 40.6
DoLa：MC2 = 63.8
提升：+23.2 points

非常明显。

### Open-ended generation

Truthfulness 提升：12–17%。

且无需：
- RAG
- finetune
- extra training

### CoT 推理

StrategyQA 提升约：1–4%
GSM8K 提升约：2%

说明：DoLa 不只增强 factuality，也帮助 reasoning。

### 延迟成本

额外 latency 仅：1.01× ~ 1.08×，几乎可忽略。

## 9. 与 Contrastive Decoding(CD) 区别

CD：大模型 − 小模型

```text
Expert - Amateur
```

DoLa：同一个模型：

```text
LateLayer - EarlyLayer
```

优点：无需第二个模型、额外训练、更多显存。

## 10. 本文最大贡献（记忆版）

一句话总结：

> 作者发现 factual knowledge 在 Transformer 后层逐渐形成，
> 因此通过“最终层 − 早期层”的 decoding，
> 放大真实知识，从而减少 hallucination。

核心创新：

1. layer contrast for decoding
2. dynamic layer selection via JSD
3. 无训练提升 factuality

## 11. 局限

作者承认：

1. 只能利用模型已有知识，错误预训练知识无法纠正
2. 不使用外部知识，无法替代 RAG
3. 主要优化 factuality，对 alignment/RLHF 未验证

## 12. Take-away（一分钟复习）

问题：LLM hallucination

观察：事实知识在后层形成

方法：DoLa = LateLayer - EarlyLayer

选层：argmax JSD

解码：softmax(log(q_N / q_M))

效果：显著提升 factuality

无需：fine-tune / RAG / extra model
